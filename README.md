# Data - Basic-Materials

It is a project wrote in Python to obtain the historical market or financial data from Basic Materials, and financial instruments (Bonds, Stocks, ETF, etc)

## Requirements
The code is using a few libraries

```
pip install pandas
pip install matplotlib
pip install yfinance
```


> [!NOTE]  
> The project was developed as a hobby during my free time.

> [!TIP]
> You can use or modify the code as you wish.



Part of the project cover fields like Data Analyst or Data science, figuring out how to get the financial information of the assets and how clean, process and present the Data to get a better approach to it.
